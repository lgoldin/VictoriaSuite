Pasos para correr Victoria Suite:

1) Bajarse Visual Studio 2017 (VS2017)

2) Clonarse el repositorio de Victoria 

3) Una vez instalado el VS2017 hacer lo siguiente 

	3.1) Tools-> Extensions and Updated -> Online (buscar "Wix") -> Instalar Wix Toolset Visual Studio 2017 Extension

	3.2) Instalar Wix Toolset para windows : http://wixtoolset.org/releases/

	3.3) Instalar WAX (Wix editor) : https://marketplace.visualstudio.com/items?itemName=TomEnglert.Wax
	
4) Boton derecho en el proyecto "Victoria.DesktopApp" -> Add.. -> Reference -> COM -> Microsfot Excel Library -> Aceptar

5) Compliar y Correr proyecto
